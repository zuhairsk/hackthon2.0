# DebugQuest Frontend

- Vite + React
- TailwindCSS v4
- Framer Motion
- Vanta.js + Three.js
- Monaco Editor

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview build

Set backend to run at http://localhost:5000. Vite dev proxy forwards `/api`.
